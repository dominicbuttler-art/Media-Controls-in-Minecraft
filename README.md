# Control your media from within Minecraft
### Main feature 
This is a simple mod that adds buttons to the pause screen to control media being played on your computer, and also adds a Now Playing label to the pause menu for you to see what is currently playing.
![The main interface](https://cdn.modrinth.com/data/cached_images/8a0e08abce29049aced413e9a3aa7174c0dbea30_0.webp)
### Now playing toast
Also adds an option to show a toast when ever the media changes, this can be toggeled on and off with ModMenu.

![Now playing toast](https://cdn.modrinth.com/data/cached_images/050bb33ff85b8ead3e3fc69ac87dd78777fbbe24.gif)
### Customise where the interface is
Allows you to control where the interface is using ModMenu.

![Custom position example](https://cdn.modrinth.com/data/cached_images/08e677aba4d39bf241c2a06cb5c5551ca30b402c_0.webp)

### Other features
- Can mute in game music when media is playing
- Works without any addtional set up
- Will also work with YouTube videos allowing you to play and pause the video from the main interface
## This mod will work with most modern media player apps
### Verified apps that this mod will work with
- Windows Media Player, not Windows Media Player legacy
- Spotify
- YouTube and Youtube Music
- Amazon Music
- Apple Music
- Google Chrome
## As of right now this mod WILL ONLY WORK ON WINDOWS DEVICES. Will not work on Linux and MacOs
Planning to add linux support in the future
